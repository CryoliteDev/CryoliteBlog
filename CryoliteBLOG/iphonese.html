The NEW iPhone SE

Apple's latest iPhone is only $399

Posted by Muhammad Azhar on May 11, 2020

Apple unveiled a new low cost iPhone this April, the iPhone SE which has the same design as the iPhone 8 that was released in 2017. Unlike the original iPhone SE this one packs the processing power of the A13 Bionic chip which is used in the latest iPhone 11 lineup. Like the original iPhone SE, which was released in 2016, this one also is the Apple’s most affordable price starting at $399.

Since the iPhone SE has the same design as the iPhone 8, so not surprisingly it still has the thick top and bottom bezels. The top bezel contains the front facing camera which is 7-megapixels which supports the Portrait Mode but lacks the Slofies which is slow motion selfies. The bottom bezel contains Touch ID home button which was discontinued back in 2017 with the release of iPhone X which uses Face ID. 

The rear of the iPhone SE has a single 12-megapixel camera which has an f/1.8 aperture, support for Portrait Mode, Portrait lightning, slow-mo video recording and optical image stabilization. However, it lacks the night mode which is covered up by the Smart HDR instead. It can also record video at 4K at 60 frames per second with optical image stabilization. Also, the whole rear side of the iPhone SE is built with glass with color matching aluminum band.

The iPhone SE contains the latest A13 Bionic chip which is the fastest chip in a smartphone till this day according to Apple. The A13 Bionic is an 8-core chip which can perform up to 5 trillion operations per second. This is the same chip that is used in Apple’s latest lineup up of iPhone 11 which has a starting price of $699, almost double the price of the iPhone SE.  Moving on to the battery life, the iPhone SE last about the same as the original iPhone 8. It last for about 13 hours with video playback and up to 40 hours with audio play back. It can be charged up to 50% in 30 minutes with 18W fast charger. The iPhone SE is also capable of wireless charging.

When it comes to configurations of the iPhone SE it comes with 64, 128 or 256GB storage options with all containing 3GB of RAM. It also has multiple color options such as black, white and (PRODUCT) RED. With all featuring the IP67 water and dust resistance so it can withstand splashes and
brief submersion in water.

Main Features: 
  • $399 starting price
  • iPhone 8 design
  • 4.7-inch display
  • A13 Bionic chip
  • Touch ID
  • Comes in Black, White, and PRODUCT(RED)
  • 64, 128, 256GB with 3GB RAM
